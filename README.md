### ✨ 주요 기능 (Key Features)
* **사원 등록**: 신입 사원 정보를 DB에 안전하게 저장합니다.
* **로그인 시스템**: 실시간 비밀번호 대조를 통한 보안 인증을 제공합니다.
* **통합 대시보드**: 사내 공지 및 사원 관리 기능을 수행합니다.
